# Board Game Center - desktop

**Preview file:** `designs/previews/board-game-center-desktop.png`
**Spec SSOT:** `designs/specs/board-game-center.md`

## Layout

Landscape 16:10 Android tablet home screen. A slim white-grey header sits above a mineral slate background. The content is a single three-slot cabinet shelf that spans the first viewport: Gomoku, Xiangqi, then Othello. Wide breathing room remains below the shelf, not extra card sections.

## Modules

- Header: a compact square seal with “棋局”, title “游戏中心”, three small inlaid installed markers, and one right-aligned dark teal import button.
- Section line: “本地棋局” and a quiet “3 个本地游戏” count.
- Three shelf slots: each is a distinct physical mini-board, a launchable badge with text, name, version, and board dimensions. The whole slot is the touch target.

## Visual Details

Use `#DCE2E2` canvas, `#F8FAF8` raised surfaces, `#B8C3C4` inset shelf slots, `#1D2B2F` ink, and `#164C5B` primary actions. Each game is differentiated by its actual board: maple 15x15 with black and jade stones; broad Xiangqi lines with brass round pieces and red accents; deep `#245E52` Othello grid with ebony and bone discs. Noto Sans SC is the UI face; Songti-style Chinese is reserved for game names. The cabinet shelf is the only memorable visual device. No gradients, no neon, no generic dashboard cards.

Use a cool mineral blue-grey background #DCE2E2 with a refined white-grey top app bar. Noto Sans Chinese UI typography with Songti-style titles. Quiet, tactile, premium, daylight, accessible contrast, flat layout with subtle object shadows.
