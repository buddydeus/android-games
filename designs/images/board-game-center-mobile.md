# Board Game Center - mobile

**Preview file:** `designs/previews/board-game-center-mobile.png`
**Spec SSOT:** `designs/specs/board-game-center.md`

## Layout

Portrait compact adaptation, 9:16. The header remains one line with seal, title, and a compact labelled import action. One-column shelf slots stack with comfortable vertical gaps. Each slot keeps the mini-board on the left and game metadata on the right; no horizontal scrolling.

## Modules

- Compact header and import action.
- “本地棋局” section label and installed count.
- Three one-column shelf slots retaining their game-specific board materials and launchable status.

## Visual Details

Use the same final tokens and typography roles as desktop. The shelf signature becomes three independent shallow recesses rather than shrinking every board into tiny equal tiles. Touch targets are visually generous; text wraps before truncating.

## Image Prompt

Create a high-fidelity Android app UI mockup, 9:16 portrait, for a Chinese offline board-game center named “游戏中心”. Use a cool mineral blue-grey #DCE2E2 background and a compact white-grey header with a small square seal and a dark teal labelled import button. The main content is a vertical cabinet shelf of three generous shallow inset slots, each showing a tangible mini-board on the left and clean Chinese game metadata on the right. The three games are Gomoku with maple wood lines and black/jade-white stones, Xiangqi with warm wood intersection lines and brass/cinnabar round pieces, and Othello with a deep green grid and ebony/bone discs. Include restrained Chinese labels 游戏中心, 本地棋局, 五子棋, 象棋, 黑白棋. Noto Sans Chinese UI typography, Songti-style game names, accessible contrast and 48dp-feeling touch controls. Quiet daylight material palette, subtle object shadows. No gradients, no neon, no carousels, no bottom navigation, no marketing sections, no stock imagery.
